Capstone Project

AWS Visual Configuration App

Our client uses Amazon Web Services quite a bit. He noticed that in doing so, he has to navigate among various AWS consoles to see all the desired information. He had an idea to visualize all of a user's AWS instances and statuses of those instances in one convenient place. And that's my team's project. The user logs into our app with their AWS credentials, and sees their instantiated AWS services visualized on one dashboard, along with the status of those instances, and the connections among instances.


Tech Stack: C#, .NET CORE, AWS SDK, Visual Studio, SyncFusion Control Library


Currently supported AWS Services: EC2, Load Balancer, ECS, RDS, S3, Route 53


Contributions: RDS Handler, ECS Handler, ECS Visualization, Documentation, Refactoring
